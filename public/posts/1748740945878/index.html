一、当期主题
#

本期James老师的核心观点围绕着长期投资、忽略市场噪音、理解资本主义本质、以及平衡财富与人生。他强调投资者应专注于如纳斯达克100指数基金（QQQ）这类优质被动型ETF，并具备长期持有的耐心和乐观心态。同时，他也深入分析了资本主义制度对不同阶层的影响，以及如何利用资本市场实现个人和家庭的财务自由，并探讨了财富积累与人生幸福、家庭关系、子女教育、父母养老和财富传承等深层议题。

二、简报内容
#

市场波动与投资心态
#

  • 无需关注市场信号：市场纷扰（如关税、战争）与长期投资无关，市场终将上涨，接近甚至创新高是常态。投资者无需为短期信号担忧。
    • 例子： 当前市场纷纷扰扰，但已快要创新高，证明了无需关心短期信号。
  • 学习历史，理解当下：美国股市百年历史表明，虽不重复，但会惊人相似。理解历史有助于坚定长期投资信念。James老师基于其40年市场经验和对200年美国市场历史的理解分享观点。
  • 乐观与耐心：投资者应永远极度乐观，不受外界悲观情绪和恐怖言论影响。“不听不看就不存在”。财富值得等待，投资需要忍耐。
  • 技术分析无效：认为技术分析有效的人尚未真正入门投资。市场波动有时如“神经病”，无需随之起舞。

核心投资策略与工具
#

  • 聚焦纳斯达克100指数基金 (QQQ)：这是当前全球成长最快的被动型ETF，适合年轻投资者加速资产增值。James老师提供了全球各地可投资纳斯达克100指数基金的资料链接。
  • 学习资源：推荐观看系列影片（如00451“价值投资讲座”为新生必修课，以及398, 399, 400等），结合当时市场情况学习，方能扎实掌握。
  • 小资族起步：建议留足半年到一年的紧急备用金后，再开始投资QQQ或00662等。有钱就买，不要择时，市价买入。

资本主义的本质与应对
#

  • 资本主义的系统性陷阱：James老师引用一位YouTuber的观点并表示认同，资本主义在设计上对富人有利，体现在：
    • 税收机制：投资收益（如美国Long-term Capital Gains，单身4万/夫妻8万以下免税，之上税率15%-20%）税率远低于劳动收入税率（最高可达37%），对长期投资者有利。台湾甚至没有资本利得税。
    • 中产阶级三重风险：工作、公司股票、公司附近房产高度绑定，一旦公司出问题，可能同时失去工作、股票价值和房产（或面临还贷压力）。建议：公司股票及时卖出转投QQQ，房子等财务自由（“买房如买拖鞋”）后再买。
    • 借贷差异：富人借钱变富（投资），中产阶级借钱变穷（消费）。
    • 收入增长差异：体力劳动收入难以追赶通胀和资产增值速度。
    • 后代发展差异：中产后代依赖“内卷”（补习、名校、花几十万挤进私校），富人后代可追求兴趣，快乐成长。
    • 金融泡沫影响：中产阶级在泡沫破裂时因撑不住而被迫低位卖出资产，富人则有钱有机会抄底。
  • 利用资本主义：尽管存在陷阱，但资本市场是普通人拉平财富差距、分享社会进步红利的重要途径。
    • 国家层面：如果国民都成为资本家，持有全球优质公司（如台积电、苹果）的股票，就能享受全球化红利，而将生产的负外部性（如污染）“转移”。国民应思考：为拥有工厂而自豪，还是为拥有工厂的股票而自豪？
  • “借一亿”思维实验：若有人愿以5%利息借你一亿（任何货币），40年后归还，你敢不敢借？敢借并懂得如何运用的人具备富人思维，因为这笔钱通过投资可以产生远超利息的回报，甚至可以立即退休。关键在于你是否有信用或资产去借到这笔钱。
  • 川普“大儿美法案”启示：为新生儿每年投资1000美元，从小培养理财观念和长期复利意识，是国家软实力的体现。这比直接发补助更有意义。

人生、家庭与财富观
#

  • 超越“成本”思维：恋爱、结婚、生子不应视为“成本”计算。拥有长远财富眼光（如100万40年后变10亿），钱是用来花的，家庭是用来分享爱的。
    • 例子： 如果只想着眼前的100万不够花，就错失了未来成为10亿富翁享受生活的可能。等到60-70岁，有钱也未必能弥补年轻时的遗憾（如生儿育女）。
  • “Die with Zero”理念：人生应尽情体验，而非守着巨额财富离世，变成“一无所有，只剩下钱”的悲哀。钱要花在刀刃上，创造美好回忆。
  • 夫妻财务智慧：
    • 经济安全感：薪水较高一方应主动将部分资金（如15万薪水拨5万）转给对方账户，给予对方经济自由和安全感，避免“事事报备”或“伸手要钱”。
    • 共同目标：夫妻理财观念一致，共同投资，不为金钱计较（如谁洗碗、谁倒垃圾）。家要有爱。
    • 先修自身：在家庭中，先让自己“值得被爱”。
  • 成为家庭的“大树”：通过长期投资，每个人都有潜力（几十万变几亿，几百万变几十亿）成为家庭的经济支柱，为家人提供荫蔽。
  • 父母养老：
    • 健康最省钱：与其担心养老金是否足够，不如多关心父母健康，做好血压血糖管理，按时服药，定期体检（如眼底黄斑部病变、关节、颈动脉支架等）。父母健康，子女负担就轻。
    • 费用预估：若父母完全不能自理，请看护加基本开销，每月可能需8-10万台币。James老师母亲10年花费约1000万台币。
    • 及早规划：子女30岁时父母约60岁，真正大额花费可能在80岁后。每月为父母定投1-2千元，20年后也能积累可观养老金。

投资实践与风险管理
#

  • 简化投资：做好资产配置后即可“自动导航”，无需每日看盘或分析资产表现。每年花5分钟做一次平衡调整即可。
  • 风险认知：
    • “市场归零”测试：任何投资决策前，问自己“如果明天这项投资归零，我能否承受？甚至三年不能卖股票，我能否活下去？”如果不行，则风险过高。
    • 流动性风险：某些市场或特定情况下（如2022年部分大陆ETF），可能出现想卖卖不出的情况。
    • 杠杆风险：融资（如股票账户的margin）风险极高，市场下跌可能导致爆仓甚至负债（如股票跌到80万，但融资100万，需倒欠20万）。个人信贷（有稳定收入前提下，用于投资）风险相对可控，因其不与股票市值直接挂钩强制平仓。
  • 学习的本质：不仅要学什么有效，更要学“这些东西都没有用”（如技术分析、经济分析、景气循环灯号）。这是在了解这些工具之后得出的结论。

三、问答环节
#

CADRE
#

  • 分享：观察到一些临近退休、无稳定收入的人想借钱投资，但他们对投资理念和风险承受能力可能不足。建议这类人群不要轻易借贷，应先用自有资金打好基础，内心足够强大后再考虑进阶操作。
    • James老师评论：非常赞同。投资者需先用自有资金经历市场波动（如10-15%回调），确认自己能承受后再考虑借贷。对于退休人士：
      • 信贷（个人信用贷款）：若无稳定收入偿还，市场下跌时会面临被迫卖股还债的风险，不建议。
      • 房贷：理财型房贷（只还利息，如美国HELOC）或许可以考虑；但需偿还本息的房贷则不建议，因金额较大。
      • 核心原则：任何借贷投资，都要思考“若股票跌至零，贷款依然要还，我能否承受？”

Sams
#

  • 分享1：作为两年多的听众，他将个人、私募基金、信托、保险产品等逐步转入QQQ和国内纳指ETF。因过于乐观，在2023年底至2024年初使用了融资（股票账户透支买入），几乎1:1杠杆。后遇“关税大戰”及国内ETF市场流动性问题（如513100），部分ETF跌停无法卖出，被迫在次日大幅平仓，去年利润大幅回吐。他父亲账户无杠杆、定期投入，反而创历史新高。深刻体会：赚钱不应急，慢即是快，安全更重要。个人现金流和未来收入预期对使用杠杆至关重要。同时，恐慌中也清掉了香港账户的QLD，错过了后续上涨。这次经历让他想起2015年A股股灾，也是融资爆仓，连续三天卖不掉。
    • James老师评论：Sams的经历非常典型。市场存在流动性风险，融资风险巨大，可能导致爆仓甚至倒欠券商。反复强调“市场归零测试”的重要性。不要考验人性。
  • 分享2：投资能让人掌握自己命运。他经历过公司被收购、再创业后公司又被收购（卖给后暴雷的碧桂园并卷入官司）、以及亲手关闭不达预期的分公司（与老板散步三小时决定关闭重庆分公司，员工次日即被N+1遣散）。因为有投资作为后盾，面对这些变故时更为从容。而没有投资的同事（如一些分公司老总）则对未来感到迷茫。感悟：无论顺境逆境，都要坚持持有好资产。
    • James老师评论：所有事物皆有意外（公司倒闭、房产损毁）。投资要先想风险，工作、个股都有高风险。QQQ也会大跌。只投资15-20-30年不用的钱。退休人士需预留约15年现金生活费。
  • 问题：James老师曾建议70% QQQ / 30% 现金，并在下跌年份配置2%杠杆基金（如QLD）。他想确认：
    1. 是否可以将中国内地和香港账户视为一体，同步按70/30配置？
    2. 因内地资金转出麻烦，是否应在香港配置更多杠杆基金？
    3. 若年底市场下跌，配置2%杠杆基金是按总资产的2%还是仅香港资产的2%？
    • James老师回复：
      1. 初学者或对波动敏感者，70/30（QQQ/现金）甚至60/40是稳妥的起点，无需急于配置杠杆基金。因为即使整体跌幅不大，杠杆基金本身的剧烈波动也可能引发焦虑（只盯QLD跌5%，不看现金不跌）。
      2. Beta可以从较低水平（如0.6-0.8）开始，随经验和资产增长逐渐提高。
      3. 关于QLD的2%配置：可以按香港资产或总资产计算。但需注意，杠杆基金通常不能用于银行质押（如汇丰银行）。若QLD占比过高，可能影响未来质押融资计划。

DVA
#

  • 分享：听James老师课如同学习人生哲学，投资是与人性做斗争。经历过近期市场回调（老师所说的“打嘴巴”行情），虽然想坚持，但还是害怕并清仓了一半，包括一些Nvidia等个股。
  • 问题：他的Roth IRA账户金额较小，临近退休。想知道若配置成60% QQQ + 10% TQQQ + 30% 现金，或者70% QQQ + 20% TQQQ + 10% 现金，使整体Beta值在0.8左右，是否足够稳健且能兼顾增长？
    • James老师回复：60/10/30 (QQQ/TQQQ/现金) 这种Beta 0.8左右的配置是很好的。Beta会随杠杆部分增长而上升。对怕震荡的人来说，初始Beta 0.6-0.8即可。

Yao
#

  • 分享：近期看新闻（马斯克在白宫工作130天后离开，期间压力巨大甚至依赖药物和游戏解压）和电影（《奥本海默》中主角的重负与无奈），感悟到“欲戴王冠，必承其重”。普通人若无此天赋和承受力，不如跟随老师理念，平淡幸福地走向富有，无需承受那些极致的压力和风险。慢慢投资，等待复利效应是很好的选择。
    • James老师评论：慢就是快。不要用你需要的钱去赚你不需要的钱。人生应追求减少遗憾，而非虚名浮利。通过QQQ等工具享受企业成果（如Nvidia），而无需承担经营者的巨大压力。

Ju
#

  • 分享：2018/2019年经朋友推荐接触James老师理念，作为股市小白深受启发，立即调整策略，用信贷资金投入并持续定投。曾参与短线交易学习（参加股票读书会），但发现辛苦且收益（ROE）不高，每天要看盘、研究美股、台股、融资变化，目标年化10-20%，资金也只敢投一部分（如10-30%），遂放弃，专注本职工作和长期投资，生活平静富足（每天早晚冥想）。向朋友同事分享理念，发现他们对为自己长期投资犹豫（觉得要等20-30年太久），但为孩子投资则较易接受，进而自己也可能参与。经历过00662从50多跌到40多的过程并坚持买入，故对当前市场波动泰然。
    • James老师评论：很多短线投资者只看单笔交易的ROI（投资回报率），而忽略了对总资产的ROE（股本回报率）。他们可能用少量资金获得高ROI，但对整体财富增长贡献甚微。例如，100万资金，用3万赚3万，ROI 100%，但ROE仅3%。而我们赚10%是赚10万。

Chen Hua
#

  • 问题1：注意到James老师近期更强调保守，建议30%现金仓位（早期为10-20%）。这是因为学员经验差异还是其他考量？他本人目标Beta 1.0，持有杠杆基金，现金较少，若要达到30%现金需减持杠杆基金。
    • James老师回复：是的，这是因为学员背景各异，许多人是新手或无法承受较大波动。30%（甚至40%）现金是“清炖牛肉面”式的基础版，适合所有人入门。有经验、能承受风险的投资者可以自行调整，选择“微辣”或“麻辣”配置。资产配置非常个人化。
  • 问题2：关于财富传承。巴菲特的遗嘱组合（90%标普500 + 10%短债）因其资产庞大而适用。普通人若后代对投资无兴趣，应如何设立一个“永久性”的信托资产配置方案？（引用梅艳芳信托案例）
    • James老师回复：
      • 首先，子女的品德和价值观教育（林则徐名言：子孙若如我，留钱做什么；子孙不如我，留钱做什么）比财富传承更重要。财富应是锦上添花。
      • 当自身资产超过年开销50倍后，可开始考虑赠与。
      • 不建议轻易设立私人非盈利基金会（Foundation），除非子女真心投入，否则可能成为负担或管理难题。
      • 信托选择：大型金融机构（如汇丰、嘉信理财 Charles Schwab）的信托服务通常比私人家族办公室更稳定（即使机构倒闭，资产也受保护，如瑞信被瑞银接管），但灵活性较低。可以设立一个例如70% QQQ / 30% 现金（或短期债券/货币基金）的组合，设定每年提取2%的规则，这样可以实现永续。关键是简单、低管理费。私人家族办公室可能包山包海，但金融机构信托更专注于资产管理和分配。

四、精彩观点
#

市场震盪都跟我沒關係了，市场可能已经纷纷扰扰扰关税来去战争还在打可是市场已经快要创新高了，所以这就是为什么无需关心市场的讯号那个都不重要，一点都不重要。 – James老师

此观点强调投资者应忽略短期市场噪音，专注长期趋势。

这个资本主义就是一个系统性的陷阱…就是对富人有钱人有利。 – James老师 (引述并认同YouTuber观点)

此观点揭示了资本主义制度下财富分配的内在机制。

你用体力赚钱永远追不上通胀了，更追不上资产生值的速度。 – James老师

此观点点明了劳动收入与资本增值的本质区别。
