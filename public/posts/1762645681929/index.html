一、当期主题
#

本期讲座的核心观点是 “投资成功的关键在于无作为”。James老师强调，投资者应该最大限度地减少决策，忽略所有市场噪音，如经济数据、国际局势、升降息等，因为这些都与长期投资无关。正确的做法是买入优质资产后就什么都不做。然而，“无作为”的前提是要对市场风险有深刻的认知，必须为可能发生的最坏情况（如资产下跌85%并持续十年）做好准备，通过严谨的资产配置和保留充足的现金流，确保自己能在这场漫长的考验中存活下来。

二、简报内容
#

课前提醒与分享
#

  • 课程编号：本次分享在YouTube上的编号为 00539。
  • 警惕诈骗：老师再次点名了一个诈骗账号（ID已改为 James.James.1），提醒大家他绝不会进行任何代客操作、收费或要求汇款的行为，所有以此为名的都是诈骗。
  • 芒格智慧影片推荐：老师强烈推荐一部关于查理·芒格人生智慧的影片（1小时23分钟）。该影片更多地探讨了人际、家庭、婚姻关系和人生哲学，而不仅仅是投资，老师认为其内容非常正确和深刻。
  • 鼓励内容创作：老师分享了学员“文烟”的例子，他使用AI工具(notebooklm)将两小时的讲座视频制作成十分钟的短片，在小红书和B站取得了很好的传播效果。老师鼓励大家利用频道的公开资料进行二次创作和传播，推广正确的投资理念。

投资心法：少即是多，无为而富
#

  • 决策越少越好：在投资中，最成功的策略就是“不做决策”。当市场恐慌、众人纷纷讨论各类宏观经济问题时，最好的应对就是忽略它们。
  • 投资的核心：投资最重要也最难的事情，就是买入后“不动”，这需要巨大的定力。任何想要操作、改变或预测市场的行为，几乎注定会降低最终的投资绩效。
  • 练习“无作为”：投资需要练习的核心能力就是“无作为”。你的感觉在市场面前是毫无价值的，甚至可以说是“垃圾”，必须做到没有感觉才能成功投资。

风险认知：为最坏的明天做准备
#

  • 市场的两面性：市场长期来看必然会不断创新高（没有最高，只会更高），这是人类社会发展的必然。但这并不意味着市场不会下跌。
  • 极端风险的必然性：投资者在一生中，必然会遇到市场从高点下跌85%的极端情况，并且这种下跌可能持续长达十年。
  • 投资者的资格：只有当你能够坦然面对资产下跌85%且十年不反弹，依然能坚守“打死不卖”的原则，并能正常生活时，你才真正具备投资的资格。如果你无法接受下跌，那就不应该投资。
  • 风险的唯一解药：对抗这种极端风险的唯一方法就是资产配置。你必须假设最坏的情况明天就会发生，并在此基础上配置你的资产，确保你持有足够的现金（例如能生活十年的现金）来度过难关。

资本主义的觉醒
#

  • 被设计的劳工：James老师指出，整个资本主义社会都在教育大众成为勤奋的劳工，而非资本家。金融体系和政府的运作模式，本质上是为了让大众努力工作、缴税、贷款买房，从而被牢牢绑定在工作岗位上。
  • 传统理财的陷阱：遵循传统的“努力工作、银行储蓄、购买保险”的路径，最终只会走向贫穷。因为储蓄会被通货膨胀侵蚀，用劳力和时间换取的金钱永远无法实现真正的财富自由。
  • 思维的转变：本频道的目的不仅是教大家如何致富，更是要唤醒大家的“资本主义觉醒”。一旦你建立了资本家的思维逻辑，即使当下没有巨额财富，你也已经走上了富有的道路，并能获得内心的解脱与快乐。

三、问答环节
#

Cady
#

  • 分享：她发现很多人存在两种极端：一种是“有钱但不知道自己有钱”，不敢投资消费；另一种是“没钱却拼命花”，对财务状况缺乏认知。她强调，要跟上老师的投资节奏，尤其是使用杠杆，必须努力学习，把知识补足，建立足够的信念和认知，否则在高点和低点都会因为恐惧而无法行动。

Chenfeng
#

  • 分享：他最近在深入学习查理·芒格的智慧，并建议大家多听智者的观点，而不是纠结于市场是否处于高点。他认为，在多头市场中，大家更应该审视自己的能力和风控，避免过度借贷。
  • 问题：查理·芒гар反复强调不要使用杠杆、不要借钱投资，这与我们频道讨论的“含辛点兵”（抵押资产借款投资）的理念似乎有所不同，请问老师如何看待其中的差异？
    • James老师回复：芒格所反对的，是那种可能导致你失去自由或破产的杠ánt。例如，一个建商将全部身家押上，结果因利率飙升而破产；或者普通人为了面子贷款买超出自己能力的豪宅，结果被贷款绑死，失去辞职的自由。芒格的核心思想是“不要因为借贷而失去自由”。而我们所提倡的借贷，前提是极为严格的风险控制：你必须假设股市跌到零并持续15年你都安然无恙，这样的钱才可以借。这要求投资者有足够的资产配置和现金流，并且不适合临近退休的人。所以，二者的核心精神是一致的，都是强调在最坏的情况下你依然能存活并保有自由。
  • 分享：他提到台湾最近有立委在推动一项针对0-10岁婴幼儿的投资计划，由政府和父母各补助4000元/月投入ETF。
    • James老师评论：这个方向是正确的，但要求父母出钱会给年轻家庭带来巨大负担，有缺陷。他建议，更好的方式是由所有上市公司每年增资1%，将这部分股份注入一个“台湾ETF池”，然后分配给新生儿，这样政府和父母都不用出钱，由资本家来为国家的未来投资，阻力会小得多。

Demo
#

  • 分享1：他认为当前的AI发展只是一个大时代的开端，远未到泡沫的程度。他愿意承受20-30%的短期震荡，以换取未来可能高达10倍的巨大收益。
  • 分享2：他分享了自己从“员工思维”到“资本家思维”的转变。过去，社会教育我们追求高收入，然后买好车、买好房，但这其实是一个让你不断为资本家打工的圈套。他分享了自己曾卖掉一台豪华车，将200万投入指数基金，这笔投资成为了他人生中最正确的决定之一。
  • 问题：最近台湾在讨论补充保费可能要根据年度股息总收入来计算，这对投资0056这类高股息ETF的投资者有什么影响？老师有什么建议？
    • James老师回复：首先，他批判了这种税制的不公，因为它专门针对依赖股息的弱势群体，而对资本利得却不征税。他认为所有所得都应公平纳税。其次，针对这个问题，他提出了解决方案：上有政策，下有对策。如果股息税或保费过高，投资者可以考虑通过“复委托”的方式，去投资美国上市的高股息ETF，如JEPQ或QQQI。这样获得的收益属于海外所得，在台湾有很高的免税额度。
    • 后续讨论补充：有学员提醒，通过复委托购买JEPQ这类产品，其股息可能会被预扣30%的税，且部分券商（如国泰）不会协助退税。因此，老师强调，在进行此类操作前，务必向自己的券商确认清楚税收政策，如果会被预扣且无法退税，则不适合投资。

Sam
#

  • 分享：他从2021年开始投资，经历过20-30%的账面亏损但坚持持有，如今不仅回本还有不错的回报，这让他对长期投资建立了信心。接触到老师的频道后，他感觉“瞬间顿悟”，仿佛从仓鼠轮中解脱出来，财务焦虑感大大降低。他也成功地将这个理念分享给了家人，他们也开始买入00662。
    • James老师评论：能帮助身边的人很好，但也要懂得“放生”，对于无法被说服的人，不必强求。投资的核心就是要有“打死不卖”的精神，无作为才会富有，而频繁操作则可能导致一无所有。

Iam
#

  • 问题1：他观察到身边有家族因为分家产而反目成仇。他困惑为何财富反而会害了家人，并请教老师如何看待家族财富传承中，第一代拼搏、第二代和谐、到第三代就可能出现分歧和矛盾的现象。
    • James老师回复：财产纠纷通常发生在继承人都很缺钱的情况下。对于他们来说，一点点不公都会被放大。最麻烦的是房产这类不易分割的固定资产。作为长辈，如果希望按自己的意愿分配财产，就必须在生前通过信托或赠与的方式做好规划。一旦人离世，一切都只能按法律条文来分配。作为子女，最好的方式就是遵循法律程序，避免纷争。
  • 问题2：他观察到，即使是自己开公司的老板，也像劳工一样不停工作、不断扩张。还有些人明明财务自由了，却不敢停下来，觉得不工作就像废人。这是否是资本主义教育的洗脑？
    • James老师回复：一个人只要每天都需要去上班，哪怕是自己的公司，他也依然是“劳工”，甚至承担着自有资金的风险，是双重风险。真正的资本家是完全没有责任，让钱为自己工作的。这种“劳碌命”的思维模式是几十年教育和工作环境塑造的，确实很难改变。芒格也提醒我们，不要为了工作赚钱而忽略了家庭，否则最后可能赢了财富，却输了人生。

Mike
#

  • 分享：他分享了自己78岁美国房东的故事。这位房东将自己建造的13处房产全部放入一个家族信托，由子孙后代共同管理，以此维系家族的凝聚力和财富。他对比了这种模式和老师推崇的QQQ模式，并感叹自己卖掉房产换成QQQ是多么明智的决定，让他不再为市场波动而兴奋或恐惧。
    • James老师回复：这位房东的做法是“不得不”的结果。持有大量房产，会面临难以分割、遗产税高昂等诸多问题。设立信托让子孙后代去管理，实际上是把他们变成了“劳工”，世世代代被这些房子所束缚。如果他持有的是QQQ，财富的传承会简单得多，可以通过每年的赠与轻松完成，子孙后代也无需为管理资产而操劳，可以真正地享受财富带来的自由。

四、精彩观点
#

投资要成功就是你的决策越少越好，不做决策就是最好的。 – James老师

背景：老师在开篇阐述其核心投资理念，强调在信息爆炸和市场波动的环境中，保持“无为”是通往成功的最佳路径。

你一定要假设明天就开始下跌而且会跌85%而且会跌10年，這樣子假设的情况之下你还做你的投资跟做资产配置…这样才你才可以投资。 – James老师

背景：在谈到风险控制时，老师提出了一个极其严苛的心理和财务测试，只有能通过这个“压力测试”的投资者，才算真正做好了准备。

你的感觉在市场来说都就是垃圾，所以在市场里面不要讲感觉，你应该是都没感觉才可以投资。 – James老师

背景：老师犀利地指出，投资是理性的领域，任何基于个人感觉、情绪的判断在市场先生面前都毫无价值，甚至是负价值。

在资本主义社会大家都被教育成要变成努力的劳工，银行跟政府根本就是想要把你的钱掏空，所以你们才能够认真的帮国家打工嘛。 – James老师

背景：老师对现行社会经济体系提出了深刻的批判，认为这套体系鼓励人们成为劳工，从而被系统性地“收割”。

芒格在讲的借贷这个事情就是说你会失去自由…你不要因为借贷失去自由，这个很重要。 – James老师

背景：在解释芒格反对杠杆的观点时，老师抓住了其核心——杠杆最大的风险不是金钱损失，而是可能导致个人自由的丧失。

储蓄是一个非常糟糕的事情，所以你的钱放银行你就是通膨来越薄，你怎么可能富有？不可能嘛。 – James老师

背景：老师挑战传统观念，指出在通货膨胀时代，把钱存在银行无异于资产缩水，是致富的大敌。

你只要每天需要起来去上班，就算那是你自己的公司，你也是劳工。 – James老师

背景：在回答学员关于老板是否是资本家的问题时，老师给出了一个清晰的定义：真正的资本家是让钱工作，而不是自己为钱工作。

五、总结
#

本期讲座从“无作为”这一核心投资哲学出发，为我们构建了一个完整的投资思想体系。James老师不仅强调了长期持有、忽略噪音的重要性，更花费大量篇幅剖析了投资中最关键却也最容易被忽视的一环——风险认知。他要求每一位投资者都必须直面市场最残酷的可能性，并以此为基准来构建自己的投资组合。在问答环节中，通过与学员的深入互动，讲座内容得到了极大的丰富和深化，无论是对芒格“不借贷”智慧的解读，还是对财富传承、税务规划的探讨，都充满了实践指导意义和发人深省的哲理。最终，讲座回归到一个根本性的思考：我们投资理财，最终是为了获得真正的自由，而不是被财富所奴役。
