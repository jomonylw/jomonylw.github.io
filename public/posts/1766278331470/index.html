一、当期主题
#

本期核心是介绍一款新的投资组合回测模拟App，并利用它来深入分析与比较股票质押与不质押两种退休提款策略的差异。James老师通过实际操作，展示了在不同市场（台湾 vs 美国）、不同利率和LTV（贷款价值比）条件下，质押策略的极限提款率（约1.5%-1.8%）及其风险。核心观点是：质押借款能够在长期显著增加最终资产总额，但对现金流的稳定性要求极高，提款率需非常保守；而不质押的策略虽然最终资产增长较少，但能提供相对较高的提款率（约2.7%），牺牲了资产增值以换取更高的灵活性。

二、简报内容
#

投资哲学与心态建设
#

  • 忽略市场杂讯：James老师开宗明义强调，市场的日常震荡、国际局势、央行升降息、经济数据、财务报表等，都与我们的长期投资策略无关。投资人应专注于长期持有，持续前进。
  • 选择QQQ的底层逻辑：我们投资QQQ等指数，并非因为“看后视镜”回顾其过往的优异表现，而是基于对未来的信念——相信人类的成长、相信科技的进步、相信美国的发展。QQQ是能最纯粹地代表这三大信念的投资标的。
  • 诈骗提醒：再次提醒学员注意，有诈骗集团创建了新的假冒账号（ID后缀增加N或M），James老师在Clubhouse中只有一个账号，大家需谨慎辨别。

投资组合回测App介绍与模拟分析
#

这款App由社群成员Martin开发，可以模拟不同的资产配置、质押条件和提款策略下的长期回报与风险。

  1. 台湾场景模拟 (40% QQQ, 30% QLD, 30% 现金)

    • 参数设置：初始本金100万，借款利率3%，LTV（贷款价值比）为60%。
    • 提款率测试：
      • 每年提取1.5%（1.5万）：模拟结果显示，从2000年到2025年，资产可以安全存活。
      • 每年提取1.6%（1.6万）：模拟结果显示，资产会在2009年被清盘（失败）。
    • 结论：在台湾的质押条件下，安全提款率的上限非常严格，仅为1.5%。

  2. 美国场景模拟 (40% QQQ, 30% QLD, 30% 现金)

    • 参数设置：借款利率6.5%，LTV更高（QQQ为70%，现金为95%，但QLD不可质押）。
    • 提款率测试：
      • 每年提取1.8%：模拟结果显示可以安全存活。
      • 每年提取1.9%：模拟结果显示会失败。
    • 结论：由于美国的LTV条件相对宽松，其安全提款率可达1.8%，风险承受能力略高于台湾。
    • 利率影响：当James老师将美国借款利率改为5%（早期Excel模拟的设定）后，安全提款率可以提升至2.1%，说明借款利率是影响质押策略安全性的关键变量。

质押与不质押策略的比较
#

  1. 不质押策略 (80% QQQ, 20% 现金)

    • 最高提款率：每年可提取**2.7%**的资产。
    • 结果：虽然可以安全度过25年，但到2025年，资产总额几乎没有增长，回到了2000年的初始水平。这是一种牺牲资产增值以换取较高现金流的策略。

  2. 同等提款率下的资产对比

    • 假设两种策略每年都只提取1.5%（1.5万）。
    • 不质押者：到2025年，资产增长至161万。
    • 质押者（台湾方案）：到2025年，资产增长至475万。
    • 结论：在同等低开销的前提下，质押策略能带来数倍的资产增長。投资人需要在“更高的资产增长”和“更高的当期现金流”之间做出权衡。

James老师的建议
#

  • 质押风险高：1.5%的提款率意味着你的总资产需要达到年开销的66倍，这对多数人来说要求很高。
  • 搭配高股息ETF：对于已退休人士，可以考虑配置一小部分资产（如10%-20%）到高股息产品（如QYI），以产生稳定的现金流，降低质押借款的金额和压力。
  • 在职人士：可以尝试质押借款，同时继续每月投入资金，情况会更为乐观。

三、问答环节
#

Martin
#

  • 分享：
    1. 介绍了回测App 0.2版的更新：可以导出和导入配置方案，避免每次重设；简化策略为“无脑再平衡”和“聪明再平衡”；增加了详细的月度流水账和半年度资产负债表。
    2. 在绩效报告中，增加了“最差年度回报”、“历史最大回撤”、“卡玛比率(Calmar Ratio)”和“痛苦指数(Pain Index)”等风险评估指标。
    3. 报告可以下载为PDF，并包含一份JSON数据，可以直接喂给大语言模型（如ChatGPT）进行白话文分析解读。
    4. 他已将App的源碼鏈接開源，方便大家共同驗證和改進。

Chen Feng
#

  • 分享：
    1. 他根据自己的Excel模型测算，认为提款率在2%甚至2.2%都应是安全的，对App模拟出的1.5%的严苛结果提出疑问，并表示愿意协助验证源碼。
    2. 分享了他制作的台版“买房vs租房”示算表，用数据证明长期来看，租房并投资，最终资产会远超买房的人，单位会是“亿”级别的差距。
    • James老师评论：James老师现场将模拟利率从6.5%调低至5%后，结果显示2.1%的提款率是可行的，证实了利率是关键变量，也说明了Chen Feng的计算在低利率环境下是成立的。老师感谢他花费大量时间制作工具和教育社群。

Ponting
#

  • 分享：他指出，在1999年到2006年期间，QLD并没有真实的交易数据，各家采用的模拟推算方式不同，会导致回测结果产生差异。这可能是不同模型之间结论有出入的原因之一。
    • James老师评论：老师认同此观点，并说明该App采用的QLD回测数据源自他早期的Excel档案，这确实会影响结果的精确性。但他强调，此工具的主要目的是让大家感受不同策略的“模糊的正确”，而非追求绝对精确的数字。

L
#

  • 问题：退休后进行Roth Conversion时，如果不做股票质押，应该如何支付税款？
    • James老師回复：有两种方式。一是直接从你的经纪账户（Brokerage Account）里的现金支付，这是最好的方式，可以最大化转入Roth账户的资金。二是直接在转换时让券商预扣税款，例如转换20万，直接扣除2万去缴税，但这样进入Roth的本金就变少了。

Andy
#

  • 分享：非常感谢老师的分享，让他和其他许多人走上了正确的财富自由之路。这种“知道终点在哪”的安心感，极大地提升了人生层次和选择的自由，远比财富本身更重要。在圣诞前夕，特别感谢老师和社群的每一个人。
    • James老师评论：我们最大的回报就是听到学员们的人生因此放轻松了。即使还未退休，但知道自己走在正确的道路上，未来有保障，这种安心感是我们最希望带给大家的。

Clark
#

  • 问题1：在模拟最差情况时（如2000年），现金部位和负债会非常接近，这样是否足够安全？
    • James老师回复：是的，模拟的结果就是“惊险飞过”，在最极端的市场下跌85%的情况下，资产刚好不会触及清盘线。这也说明了为什么提款率不能再高，2%就是那个临界点。
  • 问题2：在市场大跌、现金吃紧时，是否应该更谨慎地执行“聪明再平衡”，比如减少或暂停投入现金？
    • James老师回复：这个问题非常好，我们的确在探讨更保守的做法。一个原则是：你的现金储备若低于15年（甚至20年）的生活费，就不应该动用它去做再平衡。另一种变通方法是，不使用现金，而是卖出部分QQQ去买入QLD来进行再平衡，这样可以保全现金储备，更加安全。
  • 问题3：关于公司发的半导体股票，应该何时卖出？考虑到税务问题，是否值得等待或使用复杂的Exchange Fund？
    • James老师回复：只要持有一年以上，就适用固定的长期资本利得税率，这个税率与你的个人收入无关。所以，不要等，满一年就应该卖掉，尽早分散风险。不要碰任何复杂的金融产品，如Exchange Fund，简单就是最好的。

Shu Fen
#

  • 问题：老师上周提到的中国高股息ETF（008163），其持仓多为银行等传统行业，与以科技股为主的QQQI不同，它是否能作为替代品？
    • James老师回复：完全没问题。我们的目的是获取现金流，就像美国人吃面包，中国人吃饭，都能吃饱。不同市场的产品底层资产自然不同，只要它能稳定提供高现金流，就可以达到我们的策略目的。

五、总结
#

本期聚会通过一款实用的回测App，将抽象的股票质押策略变得具象化。James老师不仅演示了工具的使用，更重要的是揭示了不同策略背后的风险与回报权衡。质押能显著放大财富，但要求极致的纪律与耐心；不质押则更为灵活，适合对现金流有较高需求的投资者。更动人的是，多位学员，特别是来自基层的Wu Li Yun和努力拼搏的Min的女友，他们的真实故事生动地诠释了“投资可以改变命运”这一核心理念。这不仅是一堂技术分析课，更是一次关于希望、安心与人生觉醒的深度交流，再次坚定了社群成员长期持有、简单投资的信念。
