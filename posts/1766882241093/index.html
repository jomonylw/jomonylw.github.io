一、当期主题
#

本期内容深入探讨了资本主义制度的本质。James老师指出，资本主义社会就像一个巨大的迷宫，其规则和教育体系在设计上倾向于培养服务于资本家的劳工，使大多数人迷失其中。然而，这个制度也为普通人提供了阶级流动的机会。那些掌握了正确投资哲学（如长期持有优质资产）的“觉醒者”，就如同拿到了迷宫的导航系统，能够利用资本主义的规则，从被剥削的劳工转变为享受资本主义成果的资本家，最终实现财富自由。

二、简报内容
#

投资心法与市场噪音
#

  • 核心原则：市场震荡、国际局势、市场分析、财务报表、经济局势都与长期投资者无关。投资唯一需要的就是买对资产并有耐心长期持有。
  • 投资者的心态：投资者应永远极度乐观，面向阳光，坚信市场终将上涨。
  • 精神富足：真正的快乐源于精神的愉悦，而非物质的堆砌。当财富自由后，人应该追求精神上的永恒快乐，而不是通过消费来填补空虚。投资带来的内心安定是精神快乐的重要来源。

生活与健康关怀
#

  • 老年人听力问题：听力衰退是老化现象，但不能忽视。一旦发现父母有重听迹象，应尽早佩戴助听器。
  • 原因：大脑会忘记长期听不到的声音频率，导致即使声音放大也无法理解其意义。越晚佩戴，适应过程越痛苦，效果越差。听力障碍还会影响社交，可能导致老人痴呆。
  • 建议：可以带父母去Costco（好事多）等地方做听力检测并选配合适的助听器，如苹果的AirPods Pro 2也具备辅助听力功能。

投资实践与工具
#

  • 杠杆基金的配置：不应为了达到某个比例（如30%）而勉强在应税账户（Brokerage Account）中购买杠杆基金。
  • 正确做法：在免税账户（如Roth IRA）中根据资金情况持有，哪怕只有5%或10%，让其自然成长。配置杠杆基金的初衷是为了留出现金，而不是单纯追求高绩效。在震荡市中，持有现金（Beta值为0.7）的策略表现并不差。
  • 利用AI工具查找过往内容：James老师介绍了如何使用社区成员（如陈风）开发的AI工具（C.AI Gemini）来搜索过往视频中的特定内容。用户只需用自然语言提问，AI就能定位到相关的视频集数，解决了信息检索的难题。

资本主义的迷宫与导航
#

  • 制度的二元性：资本主义制度是为维护资本家地位而设计的，其整个教育体系（从幼儿园的作息铃声到高等教育）都在培养守时、听话的劳工，让他们习惯于工厂化的流程，从而无暇思考。
  • 资本家的“导航”：资本家对内（对子女）和对外（对公众）的教育是两套体系。他们自己运用的资本积累机制，对公众却宣传成需要频繁操作的“赌场”。
  • 觉醒者的道路：正确的投资哲学（长期持有、打死不卖）就是普通人走出迷宫的“导航系统”。虽然这个道理很简单，但由于多数人已被深度洗脑，很难相信和执行，这也是资本主义制度能持续运行的原因。觉醒者需要认清制度中的“剥削迷魂药”和“智慧大力丸”，做出正确的选择。

三、问答环节
#

Catherine
#

  • 分享：Catherine分享了自己从物质匮乏的童年走来的心路历程。因为从小缺乏，内心有一种匮乏感，让她很小就学会看人脸色，但也决心要成为一个有能力选择、有钱的人。在成长路上，她遇到了很多贵人，特别是James老师，让她明白了人生可以通过正确的方法和方向，踏实地走向目标。她观察到，很多和她相似背景的人不是不努力，而是认不出机会，不相信自己能改变。她感恩现在有机会在这里和大家一起学习，并愿意帮助那些对投资有疑惑、想开始却不知如何着手的朋友，因为她深知被理解和耐心陪伴的重要性。她相信，人无法选择出身，但可以选择人生的方向。
    • James老师评论：老师非常赞同Catherine的分享，并表示自己有时也会有强烈的表达欲。他分享了自己与Catherine不同的成长背景和个性，承认自己更偏向于“教导”而非“倾听”，有时这并不能真正帮助到对方。他认为，倾听他人的故事如同多过一种人生，非常有价值。他强调，每个人都有自己的故事，学会倾听非常重要。对于Catherine提到的“有福报才能听到并相信”，老师也表示认同，很多人即使听了也不会相信。

Tina
#

  • 分享：Tina是第一次参加Clubhouse，她从去年10月开始接触CLEC频道，目前听了近100期。最大的心得是内心“松了一口气”，对未来的焦虑感消失了。她认识到只要走在正确的投资道路上，自己和孩子的未来都无需担忧，孩子可以追求自己的兴趣，人生会变得很轻松。这种心态也让她在工作中能更有选择性，敢于拒绝不合理的要求。她的世界观、人生观和价值观都在向好的方向微调，也更能接纳自己不爱社交的性格。她开始思考人生的意义，并重新拾起对世界史、国际政治和哲学的兴趣。
  • 问题1：关于杠杆基金的配置。她目前是70%资产和30%现金的配置，未来想慢慢转成40%杠杆、30%指数、30%现金。她想知道，是否应该等待市场下跌5%或更好的时机，再将QQQ转为杠杆基金，因为震荡市中杠杆基金的磨损较大。
    • James老师回复：资产配置的调整不应该等待时机，而是“当下、立即、马上”执行。投资中最忌讳的就是“等待”和“犹豫”。谁也无法预测市场未来的走势，可能下周就连续大涨。既然决定了要配置杠杆基金，就应该一次性做到位，不要等待。
  • 问题2：关于退休提领率。她的一位亲戚也在用同样的方法投资，但不确定退休后提领2%还是3%更安全。
    • James老师回复：3%甚至5%的提领率都是可以的。现在有了高股息产品（如QQQY），可以灵活搭配。例如，70%的QQQY（股息率约12%）就能提供近10%的年现金流。通过配置不同市场的高股息ETF，可以轻松实现较高的提领率，这已经不再是一个难题。
  • 补充建议：James老师建议，家庭成员应尽量生活在一起，保持紧密联系。即使因为工作等原因分开，也应该安排固定的时间相聚，比如每年或每季度。这对于家庭关系和孩子的成长至关重要。

Wendy
#

  • 问题：Wendy跟随老师投资两年多，资产增长良好，预计三年后可以退休。她用投资计算器预测，按12%年增长率，20年后资产能从200万增长到1900万。她自己相信这个结果，但孩子质疑市场的多变性，问她如何能确定（for sure）。她想问老师，如何让自己更坚定地相信这个长期结果。
    • James老师回复：这个结果是“for sure”的。短期市场（15年以内）确实不确定，但长期（超过15年）来看，人类市场从未让投资者亏损，历史最低年化回报率也有6%。投资的本质是投资人类的进步，而人类社会几千年来一直在进步，从未停止。未来20年人类社会退步到石器时代是不可能的。即使是被称为“失落三十年”的日本，如果投资者持续投入并再投资股息，最终也是获利的。所以要相信人类的进步，不要被人云亦云的错误资讯（如“失落的XX年”）所迷惑。

Lucy
#

  • 问题1. 股票质押借款需要还吗？她理解质押的逻辑，但不明白如果一直借下去，最终该如何处理这笔债务，因为“借钱总要还”。
    • James老师回复：股票质押借款与房产抵押不同，是“不用还”的。因为股票是流动性高、价值增长的安全资产，银行不担心坏账（可随时平仓），所以不要求还本金。你还钱反而会降低银行的业绩。这笔债务可以一直带到生命的终点。
  • 问题2. 人去世后，质押的债务如何处理？
    • James老师回复：有几种方式：1. 将资产和债务通过信托（Trust）一同传承给继承人，继承人继续质押。2. 如果没有信托，继承人可以用自己的资金或通过质押自己的资产借钱，先还清逝者的债务，再继承全部资产。3. 如果继承人无力偿还，银行会卖掉部分遗产来偿还债务，剩余部分再由继承人继承。无论哪种方式，都比生前卖出资产要划算。
  • 问题3. 在加拿大，不同账户（RSP, TFSA, Non-registered）中，如何进行资产再平衡，是按各账户比例还是总资产比例？
    • James老师回复：应按总资产比例来看。初期，由于免税账户额度有限，可以把所有免税账户都投入杠杆基金，此时杠杆基金占总资产比例很低，无需再平衡。直到杠杆基金的总市值达到目标比例（如30%）以上时，才需要开始考虑再平衡。
  • 问题4. 券商提供的未投资现金2%的利息，是否可以替代现金管理工具？
    • James老师回复：当然不是。像BIL或类似短期国债ETF这样的现金管理工具，可以提供3.5%或更高的利息，远优于券商给的2%。

Shawn
#

  • 分享：Shawn分享了国内银行理财经理的一个案例。一位理财经理为了冲业绩，向一位大客户推荐了一款赎回费率较高（0.2%）的新理财产品。客户出于信任购买了700万人民币，年后赎回时产生了1.4万元的费用，导致客户投诉，双方关系破裂。他从中看到三点：1. 国内理财产品条款复杂，信息不透明。2. 银行理财经理的专业水平参差不齐，即使是这位理财经理自己投资美股，也是短线操作，赚了20%就跑。3. 老师说的“你的解药可能是别人的毒药”非常正确，推荐投资时必须考虑对方的风险承受能力。
    • James老师评论：老师感谢他的分享。他指出，理财经理与客户存在利益冲突，这是人性。但不应一概而论，好的理财经理应该着眼于长期，将客户“养大养肥”，而不是为了短期佣金“宰了客户”。通过提供真正有价值的建议，帮助客户资产成长，自己的佣金收入也会水涨船高。作为投资者，则需要具备分辨能力，自己要懂。

Doris
#

  • 问题：Doris分享了她对QQQY的观察。她发现持有QQQY后，其成本基数（Cost Basis）会因收到“资本返还”（Return of Capital）而降低，这部分在税表上体现为Nondividend Distribution，无需缴税。她想确认，这种成本基数的不断下降，会不会损害净资产？她理解净资产是看当前市值，但想确认这种操作的长期影响。
    • James老师回复：Doris的观察完全正确。QQQY的设计就是通过“期权收益”+“部分本金返还”来创造平滑、稳定的现金流，这对于退休生活至关重要。虽然成本基数会降低，但QQQY本身持有80%的QQQ头寸，资产净值依然会随市场长期上涨。这种机制对投资者是好事，因为它能提供稳定的现金流且具有税务优势。我们反正不卖，所以成本基数的变化没有影响。

Tao
#

  • 问题：关于资产传承。他想咨询，当资产达到何种规模时，需要考虑设立遗嘱（Will）或可撤销信托（Revocable Trust）？具体应该如何操作？
    • James老师回复：在美国，个人遗产税免税额很高（约1400万美元），但我们投资者的资产未来都会超过这个数，所以应尽早规划。
    1. 赠与：首先要利用好每年的赠与免税额（约2万美元/人），定期将资产转给子女。
    2. 指定受益人(Beneficiary)和TOD：所有投资账户都应填好受益人（Beneficiary）。房产则可以去郡（County）登记处办理TOD（Transfer on Death），指定过世后的继承人。这两项措施可以绕过耗时费力的遗嘱认证（Probate）过程。写遗嘱本身并不能避免遗嘱认证。
    3. 信托（Trust）：只有当你担心子女无力偿还你的质押贷款，导致资产被强制出售时，才需要考虑设立信托。将股票账户和质押贷款账户都放入信托，子女可以直接继承信托，同时也继承了资产和债务。
  • 问题2：如何处理赠与可能导致孩子失去奋斗动力的问题？
    • James老师回复：这取决于从小建立的价值观。如果担心，可以为孩子开设一个由你全权管理的账户（Power of Attorney），资产在他名下，但由你控制，直到你认为合适的时机。同时要持续向孩子灌输正确的金钱观，让他们明白父母只负责到大学毕业，之后需要自己养活自己。

Evan
#

  • 问题：Evan分享了每年父母从台湾来访时，他感受到的思想冲击。父母已70多岁，生活无忧，但依然每天盯盘，认为人活着就要工作，无法理解和接受他所学的长期投资理念。他感到困惑，面对无法被说服的长辈，是应该任由他们按自己的方式生活，还是应该尝试去引导他们？
    • James老师回复：改变他人是非常困难的，尤其是当他们认为自己过得很好时。老师分享了自己陪伴母亲最后十年的经历，母亲也喜欢炒股，老师并未阻止，因为那对她而言是一种“娱乐”。我们无法强行“拯救”一个不认为自己需要被拯救的人。正确的做法是，我们自己要先成为一棵可以为家人遮风挡雨的“大树”。当你自己通过正确的投资变得越来越富有，你的成功本身就是最有力的证明和说服力。那时，家人自然会看到光，并可能被影响。在此之前，顺其自然，做好自己即可。

四、精彩观点
#

资本主义制度就是一种劳工应该说资本主义的教育就是培养劳工，你资本家当然教育所有的人都是劳工啊，对你资本家最有利啊。

这个观点揭示了现代教育体系的深层目的，即为资本主义机器培养标准化的劳动力，而非独立的思考者。

在这个资本制度的迷宫里面，资本家是有一个导航系统，可是一般人没有。

James老师用“迷宫”和“导航”的比喻，生动地说明了信息不对称在资本主义社会中的关键作用，掌握正确知识的人才能找到出路。

你试着去找100个人跟他讲这个制度，讲投资哲学，讲我们的长期持有，他说你神经病了。对，这就是资本主义的目的。

这句话点明了长期投资理念之所以小众的原因：主流社会已经被消费主义和短线投机思想深度洗脑，以至于正确的道理反而被视为异端。

股票质押借钱是不用还的啦……你借钱是银行的大客户，你一还钱他的生意就掉下来了。

这个观点颠覆了传统“欠债还钱”的观念，揭示了优质资产抵押贷款的金融本质，即银行乐于向拥有增值资产的客户持续提供流动性。

我们自己要变一个大树，所以可以庇荫自己的亲人。那你就看着他，那你越来越有钱，他会学会的。

在面对家人不理解时，与其空口说教，不如用自己的成功来证明。当自己足够强大时，自然能影响和保护身边的人。

除非那个人要给你催眠啊，如果那个人都不给你就是心里抗拒，你要怎么催也没办法催眠啊。

这个类比说明了改变他人思想的先决条件是对方的主观意愿。对于不愿意接受新观念的人，任何强行的“拯救”都是徒劳的。

会念书以后不是优点，反而是缺点，这个要注意，会玩是优点。

在人工智能时代，知识的记忆和检索能力已不再是人类的核心优势。相反，创造力、好奇心和“会玩”的能力，成为了更宝贵的特质。

投资资产配置要买卖没有时机，就是当下、立即、马上。

这句话强调了在投资决策中执行力的重要性。一旦策略确定，任何的等待和犹豫都可能错失机会，应立即行动。

五、总结
#

本期聚会的核心思想是揭示资本主义社会的运行逻辑，并为身处其中的普通人指明一条通过投资实现阶级跨越的道路。James老师强调，社会的主流教育和价值观旨在将人塑造成安分守己的劳工，而真正的财富密码——长期持有优质资产——却被刻意掩盖。投资者需要做的就是屏蔽市场噪音，坚信人类社会的长期进步，用耐心和纪律来执行简单的投资策略。通过问答环节，老师进一步解答了关于杠杆基金配置、资产传承、退休提领以及如何与持有不同观念的家人相处等实际问题，再次巩固了“成为自己的大树，用结果说话”的核心理念。最终，实现财富自由不仅是物质上的解放，更是通往精神富足和人生意义探索的起点。
