一、当期主题
#

本期直播的核心在于探讨并塑造正确的投资者心态。James老师强调，成功的投资者应如同优秀的创业者一样，永远保持极度乐观，永远面向阳光，坚信市场终将上涨。投资者需要学会过滤市场震荡、国际局势、分析师预测等一切外部噪音，专注于自己能控制的资产配置和长期策略。同时，课程深入探讨了在面对杠杆投资时，如何平衡理智上的潜在收益与情感上的焦虑，最终强调心理健康应优于潜在的财务获利。

二、简报内容
#

投资者的核心心态与原则
#

  • 过滤无效信息：James老师开篇即指出，市场震荡、国际局势、市场分析、财务报表、经济局势等都属于噪音，投资者不应将时间浪费在这些无法预测或控制的因素上。任何分析要么是事后诸葛，要么是猜测，其准确性与抛硬币无异。
  • 保持极度乐观：成功的投资者心态与创业家（如伊隆·马斯克）相似。马斯克的公司虽屡次濒临破产，但他始终保持极度乐观。投资者也应如此，坚信市场长期向上，科技和人类社会将持续发展。
  • 投资者的信念：
    1. 投资者永远极度乐观。
    2. 投资者永远面向阳光。
    3. 市场终将上涨。
    4. 投资需要忍耐，财富值得等待。

重要提醒与澄清
#

  • 关于中国ETF的更正：James老师郑重致歉并更正了之前的观点，指出中国内地的“南方红利低波动50ETF”（515450）等产品存在误导性。其网站展示的收益图包含了股息再投资，而实际股价可能是下跌的。他明确建议不要投资这类产品，并表示仍在为中国内地和香港投资者寻找合适的高股息解决方案。
  • 警惕诈骗：老师强调，他本人不会在Clubhouse或其他平台主动私信任何人，也不会有任何收费群或代客理财行为。例如Clubhouse里就有 cchc.james 后面加 m 或 cec.chairman 这类仿冒ID，任何主动嘘寒问暖或要求汇款的行为均为诈骗。
  • 沟通渠道：提问应通过YouTube留言或发送Email，老师会尽量在两天内回复。
  • 无官方群组：CLEC没有任何官方群组。虽然有学员出于善意建立微信群，但存在被封禁和诈骗混入的风险。建议投资者自行组建由亲朋好友构成的封闭式小群进行讨论，但任何群组均不代表CLEC。

关于货币与汇率的探讨
#

  • 美元的霸权地位：讨论了美元作为国际储备货币的地位难以撼动。一个国家的货币要国际化，其国家的产业实力和综合国力必须足够强大，能为全球提供所需的产品和服务，而这恰恰是最难的。
  • 汇率对个人的影响：对于普通人而言，本国货币是否为国际货币影响不大。汇率主要影响的是国民的购买力。例如台币对美元贬值，意味着出国旅游的成本会更高。汇率问题是复杂的政治经济学，涉及产业、政策、历史等多种因素。

杠杆投资的理智与情感
#

  • 案例分析：一位学员来信分享了他的矛盾心理。他通过理智分析，认为通过信用贷款或房贷进行杠杆投资，可以快速积累本金，是正确的决策。
  • 情感的阻碍：然而，当他准备实际操作借钱时，却开始感到极度焦虑和烦躁，每天都忍不住看盘，市场的小幅波动都会让他心神不宁，无法安睡。
  • James老师的建议：
    • 心理健康优先：老师明确指出，如果借钱投资会让你产生如此大的心理压力，那就不要借。心理健康远比潜在的财务收益更重要。
    • 知行合一的挑战：这个案例是典型的“想致富又怕扛杠杆”的矛盾。投资如同吃药，即使是良药，如果副作用致命（如此案例中的严重焦虑），就绝不能服用。
    • 自我评估：能想到每天看盘、涨跌百分比这些细节，说明这位投资者本身性格就比较紧张，不适合承担杠杆带来的心理压力。安分投资，不借钱，是更适合他的道路。

Steven
#

  • 问题：他的儿子在美国2025年开了公司，关于报税问题感到困惑。公司有分红，不知道是该报公司税还是个人所得税，如何规划才能更省税？
    • James老师回复：这是一个税务规划问题。建议：
    1. 将大部分利润留在公司，缴纳相对较低的公司税（约18%）。
    2. 儿子可以从公司领取一部分薪水作为个人收入，将个人所得税的边际税率控制在较低的水平（如22%以内），大概十几万美元。
    3. 公司的很多开销可以用来抵税。
    4. 关于退休账户，强烈建议只放Roth 401K和Roth IRA，绝对不要放Traditional（税前）账户，避免未来转换时面临高额税率。

Winnie
#

  • 问题：她还有三年退休，计划退休后每年将Traditional IRA里的30万美元转换到Roth IRA，预计每年要缴12万美元的税，持续8年。她目前的能力只能提前存下两三年的税金，想问该如何准备这笔税款。
    • James老师回复：
    1. 不要刻意存钱缴税，而应通过资产配置和金融工具来解决。
    2. 首选方案是股票质押借款来缴税，绝对不要卖掉投资账户（Blockage Account）的资产，因为卖出会产生新的资本利得税，问题会更复杂。
    3. 你的总资产包括Blockage、Roth和Traditional账户，计算总资产的2%是否足够支付每年的税款。
    4. 如果质押借款的额度不够，可以考虑在做Roth Conversion时，一部分资金转入Roth，另一部分转入Blockage账户，以增加可抵押的资产。
    5. 最后的选择才是在转换时直接从Traditional IRA里预扣税款（tax withholding），但这样会损失一部分本金的复利增长。

四、精彩观点
#

投资者的心态跟创業者一样，永远极度乐观，永远面向阳光，市场终将上涨。 – James老师

背景：James老师将投资者的心态与伊隆·马斯克面对公司濒临破产时的乐观精神相类比，强调了信念对于长期投资的重要性。
